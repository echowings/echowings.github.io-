Set Session timeout

exec timeout command is sused to specify the timeout for exec sessions[telnet/ssh] whereas session timeout command specifies the idle timeout period for all the sessions.

1
2
3
4
5
6
7

config t
line vty 0 15
exec-timeout 10
session-timeout 10

#Check configure
sh run | be line vty
